Preguntas frecuentes

¿Cuál es la duración del código QR?

El código QR tiene una duración de 24 horas desde que es generado, por lo tanto, cada día se crea un nuevo código automáticamente para garantizar la seguridad.

¿Por qué no puedo acceder a la sección de “Análisis estadístico”?

La sección de análisis estadístico está habilitada únicamente para aquellos usuarios que son del tipo administrativo, por lo tanto todos los usuarios de un tipo distinto al mencionado no podrán acceder al contenido de esta sección.

¿Para qué sirve la sección de “Ver datos personales”?

Esta sección de la aplicación es la que puede observarse una vez que el personal de la entrada o salida de las instalaciones escanea el código QR; de esta forma el personal encargado puede validar que el usuario registrado es realmente quien presenta el código, mientras que el propio usuario puede revisar que datos personales se están proporcionando.

¿Por qué en la sección de “Editar datos personales” no puedo modificar el tipo de usuario?

El tipo de usuario no puede modificarse debido a que con base en este dato la aplicación web permite realizar ciertas acciones; además, proporciona un mayor grado de seguridad para la identificación del usuario durante el ingreso a las instalaciones.
Importante: Si por alguna razón registró incorrectamente el tipo de usuario y necesita cambiarlo, deberá acudir de forma presencial y con una identificación vigente al área administrativa de la ESCOM, para realizar el trámite de cambio.

¿Por qué en la sección de “Editar datos personales” no puedo modificar mi identificador de usuario?

El identificador de usuario es un dato que se asigna o registra únicamente una vez dependiendo del tipo de usuario, es por esta razón que no debe de ser modificado por ningún motivo.
Además, el usuario es responsable de no compartirlo con nadie y siempre recordarlo, debido a que este dato es necesario para el inicio de sesión en la aplicación.
